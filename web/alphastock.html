About This Project

下一代 A 股
智能投研系统

AlphaStock 是基于 LangGraph 多智能体框架与 DeepSeek R1 深度推理模型构建的 A 股全栈投研平台。 系统通过多个 AI Agent 并行分析基本面、技术面与市场情绪，最终通过辩论网络达成投资决策。

🧠
多智能体辩论架构
多头 Agent 与空头 Agent 同时工作，分别从基本面、技术面、情绪面建立论点，最终由裁判 Agent 综合作出投资决策，避免单一视角偏差。
📊
量化回测引擎
基于 Backtrader 框架，内置 KDJ 超卖反弹、J 线极值反转、RSI、布林带突破、KDJ+MACD 共振五种策略，支持自定义日期区间回测。
✨
Alpha 多因子选股
并行计算全市场 A 股的 KDJ、量能、ROE、市值、趋势五维因子综合评分，自动筛选符合阈值的优质标的，支持板块细分。
⚡
实时行情 + RAG 增强
接入实时 A 股数据 API，结合 RAG（检索增强生成）技术，AI 分析时自动注入最新财报、公告及行业研报作为上下文。
